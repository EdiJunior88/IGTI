E este é um parágrafo, um dos elementos mais comuns em documentos HTML.

Existem diferentes níveis de título, do 1 ao 6:

Título nível 3

Título nível 6

É possível enfatizar uma palavra com o elemento em, dar ainda mais ênfase com elemento strong, ou marcar uma palavra com elemento mark. Embora existam os elementos i (itálico) e b (negrito), eles não são mais recomendados, pois o HTML deve focar no conteúdo e semântica, deixando a estilização para o CSS.

Também podemos dermacar código fonte com o elemento code, ou demarcar entrada de teclado com o elemento kbd.
